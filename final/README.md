# 🎶 The Soundwaves Collective

Welcome to *The Soundwaves Collective* — a responsive and dynamic music showcase project built as the final assignment for a web development course.

## 🌟 Project Overview

This site honors the *greatest music artists of all time*, celebrating their artistry, stories, and cultural impact. Designed with music lovers and collaborators in mind, it brings together biographies, genres, and inspirational messages in a sleek and accessible layout.

## 🧩 Pages Included

- index.html – Home page introducing the site and featuring legendary musicians
- artists.html – Dynamic showcase of musical icons with genre-based filtering
- contact.html – Responsive contact form with client-side validation and localStorage
- references.html – Source citations for content, images, and fonts
- 404.html – Custom error page with redirect
- README.md – You’re reading it!

## 🛠 Technologies Used

- HTML5 (semantic and accessible)
- CSS3 (responsive grid, animations, PARC principles)
- JavaScript (DOM interaction, event listeners, arrays, localStorage)
- GitHub Pages for hosting
- Google Fonts (Roboto)

## ⚙ Features

- Fully responsive for desktop and mobile
- Artist cards dynamically rendered via JavaScript
- Genre filter dropdown
- Contact form with validation and localStorage backup
- Smooth animations and back-to-top button
- Custom 404 page and favicon
- Social links and fan quotes
- Fully optimized and lazy-loaded images
